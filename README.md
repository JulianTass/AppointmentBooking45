# AppointmentBooking45

Mobile-responsive appointment calendar + backend API (SQLite).

## Run locally

```bash
cd "/Users/juliantass/Library/CloudStorage/OneDrive-GenesysTelecommunicationsLaboratories,Inc/AppointmentBooking/AppointmentBooking45"
python3 backend/server.py
```

Open `http://localhost:8080/`.

## Notes

All API inputs/outputs use Australian-friendly datetime strings (`DD/MM/YYYY HH:mm`) and the backend interprets them as `Australia/Sydney`.
